  <h1 style="color: #2c3e50;">🎓 Student Management System</h1>
  <p>
    The <strong>Student Management System</strong> is a desktop-based Java application designed to assist educational institutions and administrators in managing student data efficiently. This project utilizes Java's <strong>Swing framework</strong> to provide a user-friendly interface for performing basic CRUD (Create, Read, Update, Delete) operations on student records.
  </p>

  <h2 style="color: #2c3e50;">🌟 Key Highlights</h2>
  <ul>
    <li>✅ GUI-based application using Swing</li>
    <li>📋 Add, search, and delete student entries</li>
    <li>🧠 Object-oriented programming concepts used throughout</li>
    <li>🔐 Input validation for fields like name and age</li>
    <li>📁 Easily extendable for file/database integration</li>
  </ul>

  <h2 style="color: #2c3e50;">🛠️ Built With</h2>
  <ul>
    <li><strong>Java (JDK 8+)</strong> – Core programming language</li>
    <li><strong>Swing</strong> – GUI toolkit for desktop applications</li>
    <li><strong>IntelliJ IDEA</strong> – Recommended IDE for development</li>
  </ul>

  <h2 style="color: #2c3e50;">📂 Project Structure</h2>
  <pre style="background: #eee; padding: 10px; border-radius: 5px; overflow-x: auto;">
